# Parabola Skater

一个把二次函数做成闯关体验的数学游戏。玩家通过调节函数参数来改变抛物线轨道，让小滑手顺利从起点平台滑到终点平台。

## 在线演示

试玩地址：[https://m4xzheng.github.io/ParabolaSkater/](https://m4xzheng.github.io/ParabolaSkater/)

## 项目简介

这个作品表面上是一个闯关游戏，但核心是二次函数图像的变化。玩家需要通过观察、尝试和调整参数，让抛物线轨道变成合适的形状，从而帮助角色完成关卡。

游戏目前包含两个关卡：

- 第一关：感受 `a` 的力量。玩家调节 `y = ax^2` 中的 `a`，观察抛物线的开口方向和陡缓变化。
- 第二关：移动顶点。玩家调节 `y = a(x - h)^2 + k` 中的 `a`、`h`、`k`，把抛物线顶点移动到目标区域，并让轨道和左右平台连接起来。

## 玩法说明

进入游戏后，玩家可以看到关卡画面和参数滑块。

玩法流程：

1. 拖动滑块调整函数参数。
2. 观察抛物线轨道的形状变化。
3. 点击开始运行。
4. 观察小滑手是否能顺利到达终点。
5. 如果失败，根据提示继续调整参数。

游戏会根据当前参数模拟角色的运动结果。如果轨道太平、太陡、开口方向不对，或者没有和平台连接好，角色就无法成功通过。

## 数学原理

第一关使用标准形式：

```text
y = ax^2
```

其中：

- `a > 0` 时，抛物线开口向上。
- `a < 0` 时，抛物线开口向下。
- `|a|` 越大，图像越陡。
- `|a|` 越小，图像越平。

第二关使用顶点式：

```text
y = a(x - h)^2 + k
```

其中：

- `a` 控制开口方向和陡缓程度。
- `h` 控制图像左右移动。
- `k` 控制图像上下移动。
- 顶点坐标是 `(h, k)`。

通过这两个关卡，玩家可以先单独理解 `a` 的作用，再进一步理解多个参数一起改变图像时的效果。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

运行测试：

```bash
npm test
```

构建生产版本：

```bash
npm run build
```
